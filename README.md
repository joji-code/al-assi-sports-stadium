# 🏟️ Al Assi Sports Stadium

A simple web project designed to present and organize information about **Al Assi Sports Stadium** in a clean and user-friendly way.

---

## 💡 Project Idea
This project simulates a basic digital system for a sports stadium, aiming to present stadium-related information in an organized and accessible interface.

It is part of my learning journey in web development and frontend design.
